<h1 align="center"> Simple Programming Language </h1>

## Переменные
```
a = 5
b = 10
print(a + b)
```

## Условные ветвления
```
a = 5
if(a < 10){
	print("a < 10")
} else {
	print("a >= 10")
}
```

## Циклы
```
i = 0
while(i < 10){
	print("i = "..i)
	i = i + 1
}
```

## Преобразование типов
```
input(a)
input(b)
int(a)
int(b)
print(a + b)
```
